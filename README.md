# Grocery App Example

This repository contains a small grocery shopping app with an Express backend and a vanilla JS frontend. It demonstrates basic cart functionality but is **not** a full BigBasket clone.

## Structure
- `server/` – Express backend with in-memory data
- `client/` – vanilla HTML/JS frontend

## Running the backend
```bash
cd server
npm install
npm start
```

## Using the frontend
Open `client/index.html` in your browser while the backend is running. The page allows you to add items to a cart, remove them and view the total price.
